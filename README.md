# RUST EVM TEMPLATE

This is a template for creating Rust project on EVM chains. 

## Blueprint

```
src
├── main.rs
├── abi - Holds contract abis 
│   └── ...
└── utils
    ├── constants.rs - Holds constants
    ├── dotenv.rs - Holds functions for accessing and utilizing environment variables
```
